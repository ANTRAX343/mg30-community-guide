

# Especificaciones Técnicas - NUX MG-30

## Información General

- **Dispositivo**: NUX MG-30, modelador y multi-efectos para guitarra y bajo
- **Fabricante**: Cherub Technology Co. Ltd. (NUX Audio)
- **Origen**: China
- **Software complementario**: QuickTone para edición de sonido y actualización de firmware

## Seguridad

- Usar únicamente el adaptador de corriente original
- No exponer el dispositivo a calor, líquidos o humedad
- No abrir el equipo, solo servicio técnico autorizado
- La escucha prolongada a alto volumen puede causar daños auditivos
- Cumple con normativa FCC Clase B y CE (EN 61000-6-3 / EN 61000-6-1)

## Características Principales

- Pantalla LCD de 4 pulgadas en color, resolución 800x480
- Procesamiento con dos DSP y algoritmo TSAC-HD White Box Amp
- Once bloques de señal independientes con opción de enrutado paralelo
- Efectos disponibles: WAH, compresor, noise gate, efectos varios, amplificador, IR, EQ, envío/retorno, modulación, delay, reverb
- Impulse Response con resolución 1024 y soporte para IRs de terceros
- Modelado de amplificadores para guitarra eléctrica, bajo y simulación acústica
- Ajuste de entrada con EQ de tres bandas en configuración global
- Cinco modos de salida con EQ de tres bandas
- Función de seguimiento de parámetros para conservar ajustes de efectos
- Interfaz USB para grabación, firmware y edición con QuickTone
- Funciones adicionales: looper, batería rítmica, afinador, juego Snake

## Controles Principales

- Perillas 1 a 4 para parámetros de efectos
- Perilla maestra para volumen general
- Codificador de cinco vías para navegación y edición
- Botones de Home, Global, EXP y Guardar
- Footswitches para cambio de patch y control asignable como tap tempo o activar efectos
- Pedal integrado para volumen o wah, con opción de pedal externo adicional

## Configuración Global

- Pre-EQ para ajustar la entrada con ecualización de tres bandas
- Output Mode para seleccionar tipo de salida y aplicar EQ global
- Asignación de pedal de expresión interno o externo, independiente o por patch
- Ruteo USB configurable también desde QuickTone
- Ajuste de brillo de pantalla
- Parámetro Follow para mantener ajustes entre cambios de modelos
- Opción de restaurar valores de fábrica

## Especificaciones Técnicas

- **Impedancia de entrada**: 1 MΩ
- **Impedancia Return**: 220 kΩ
- **Impedancia de salida**: 1 kΩ
- **Impedancia Send**: 1 kΩ
- **Conversor A/D**: 32 bits
- **Frecuencia de muestreo**: 48 kHz
- **Rango de respuesta de frecuencia**: 20 Hz a 20 kHz
- **Rango dinámico**: 110 dB
- **Consumo eléctrico**: < 400 mA
- **Alimentación**: 9V DC centro negativo
- **Dimensiones**: 313 x 166 x 54 mm
- **Peso**: 1.5 kg

## Contenido de la Caja

- NUX MG-30
- Adaptador de corriente
- Cable de parcheo TRS
- Conmutador dual NMP-2
- Cable USB
