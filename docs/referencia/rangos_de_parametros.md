# Rangos de Parámetros — NUX MG-30 v5.0.2

> **Firmware objetivo:** NUX MG-30 **v5.0.2**

Referencia rápida de rangos de parámetros comunes en los efectos de la NUX MG-30.

## Parámetros comunes

| Parámetro | Rango típico | Unidad | Notas |
|-----------|-------------|--------|-------|
| Drive / Gain | 0 – 100 | — | Cantidad de saturación/distorsión |
| Tone | 0 – 100 | — | Brillo/oscuridad del sonido |
| Level / Volume | 0 – 100 | — | Nivel de salida del efecto |
| Mix / Wet | 0 – 100 | % | Mezcla señal seca/húmeda |
| Rate | 0 – 100 | — | Velocidad de la modulación |
| Depth | 0 – 100 | — | Profundidad de la modulación |
| Time / Delay | 1 – 2000 | ms | Tiempo de retardo |
| Feedback | 0 – 100 | % | Repeticiones del delay |
| Decay | 0 – 100 | — | Cola de la reverb |
| Pre-Delay | 0 – 500 | ms | Tiempo antes de la reverb |
| Threshold | -80 – 0 | dB | Umbral del noise gate / compresor |
| Ratio | 1:1 – 1:∞ | — | Ratio de compresión |
| Attack | 1 – 100 | ms | Tiempo de ataque del compresor |
| Release | 10 – 1000 | ms | Tiempo de liberación |
| Send Level | -15 – +15 | dB | Nivel de envío en loop S/R |
| Return Level | -15 – +15 | dB | Nivel de retorno en loop S/R |
| Patch Level | 0 – 100 | — | Nivel global del preset |

## Notas de uso

- Los valores por defecto varían por efecto; consulta la página de cada efecto para valores específicos.
- El parámetro **Mix** en 0 equivale a señal completamente seca (sin efecto); en 100 es señal completamente húmeda.
- Para el **Noise Gate**, valores de Threshold demasiado altos pueden cortar el sustain de la guitarra.
