# Combinaciones Recomendadas — NUX MG-30 v5.0.2

> **Firmware objetivo:** NUX MG-30 **v5.0.2**

Cadenas de efectos recomendadas para diferentes estilos y escenarios de uso.

## Cadena básica (orden estándar)

```
WAH → CMP → GATE → EFX → EQ → AMP → CAB → S/R → MOD → DLY → RVB → GLOBAL
```

> En la MG-30, el orden de los bloques es fijo pero algunos pueden activarse/desactivarse individualmente.

## Combinaciones por estilo

### 🎵 Limpio (Jazz / Funk)
```
CMP (Studio Comp) → AMP (Jazz Clean) → CAB (JZ120) → MOD (CE-2) → RVB (Room)
```
- **CMP**: Ratio 3:1, Threshold moderado, Attack rápido
- **AMP**: Bass 5, Mid 6, Treble 5, Presence 4
- **MOD**: Rate lento, Depth bajo

### 🔵 Blues
```
CMP (Rose Comp) → EFX (T Scream, Drive bajo) → AMP (Deluxe RVB) → CAB (DR112) → DLY (Analog Delay) → RVB (Spring)
```
- **EFX T Scream**: Drive 30–45, Tone 55, Level 60
- **DLY**: Time 350–450 ms, Feedback 25–35%

### 🎸 Rock Clásico
```
EFX (Klon o Morning Drv) → AMP (Plexi 100 o Brit 800) → CAB (1960 o GB412) → MOD (Phase 90) → DLY (Tape Echo)
```

### 🤘 Metal Moderno
```
GATE (Noise Reduction) → EFX (EAT o Red Dirt) → EQ (10-Band, boost mid-alto) → AMP (Dual Rect o SLO 100) → CAB (1960) → DLY (Digital Delay, tiempo corto) → RVB (Room, mezcla baja)
```
- **GATE**: Threshold ajustado para cortar ruido sin afectar el sustain
- **EQ**: Ligero corte en 300–500 Hz, boost en 2–4 kHz

### 🌙 Ambient / Shoegaze
```
EFX (RC Boost) → AMP (limpio) → CAB (A212) → MOD (Flanger o Shimmer mod) → DLY (DuoTime, tiempos largos) → RVB (Shimmer, Decay alto)
```

### 🎻 Acústica Simulada
```
AMP (Optimal Air) → CAB (G-HBIRD o G-J15) → EQ (Align) → RVB (Hall, mezcla baja)
```

## Notas generales

- El **GATE** conviene colocarlo antes del EFX para suprimir ruido de alta ganancia.
- El **EQ** después del AMP permite correcciones de sala/mezcla sin afectar el carácter del amplificador.
- El **S/R** en modo paralelo permite mezclar efectos externos con la señal original.
