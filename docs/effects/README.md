## Documentation Format

Each effect includes:
- **Description** - What the effect does and its real-world inspiration
- **Parameters** - All available controls and their ranges
- **Usage Tips** - Recommended settings and applications
- **Related Effects** - Similar or complementary effects

## Quick Search

Use Ctrl+F (Cmd+F on Mac) to search for:
- Effect names
- Parameter names
- Amplifier brands (Fender, Marshall, Vox, etc.)
- Effect types (overdrive, chorus, delay, etc.)
