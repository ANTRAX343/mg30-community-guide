# RC Boost (EFX)

## Descripción
Emulación del Xotic RC Booster, un pedal de boost transparente que permite realzar la señal y moldear el tono con controles de ganancia, volumen y ecualización de graves y agudos.

## Parámetros

| Parámetro | Valores posibles | Descripción |
|-----------|-----------------|-------------|
| Gain      | 0–100           | Controla la cantidad de ganancia añadida. |
| Vol       | 0–100           | Ajusta el nivel de salida general. |
| Bass      | 0–100           | Ecualiza las frecuencias graves. |
| Treble    | 0–100           | Ecualiza las frecuencias agudas. |
