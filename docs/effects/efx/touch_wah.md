# Touch Wah (EFX)

## Descripción
El Touch Wah es un efecto de auto-wah/envelope filter que responde a la dinámica de la interpretación. Permite elegir entre varios modos de wah clásicos (Cry, VX, Full, Talk) y ajustar la sensibilidad, el tiempo de decaimiento, la dirección del barrido y el nivel de salida. Es ideal para funk, rock y sonidos expresivos que reaccionan al ataque de la pua o los dedos.

## Parámetros

| Parámetro | Valores posibles         | Descripción |
|-----------|-------------------------|-------------|
| Mode      | Cry, VX, Full, Talk     | Selecciona el tipo de wah/envelope filter |
| Decay     | 0-100                   | Controla el tiempo de decaimiento del filtro |
| Sens      | 0-100                   | Ajusta la sensibilidad de respuesta al ataque |
| Up/Down   | Up, Down                | Cambia la dirección del barrido del filtro |
| Level     | 0-100                   | Controla el nivel de salida del efecto |
