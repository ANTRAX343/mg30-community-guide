# Red Fuzz (EFX)

## Descripción
El Red Fuzz está inspirado en el clásico fuzz Face, un pedal de fuzz legendario utilizado por guitarristas como Jimi Hendrix y David Gilmour. Es conocido por su sonido saturado, cálido y con gran respuesta dinámica, ideal para solos expresivos y texturas psicodélicas.

## Parámetros

| Parámetro | Valores posibles | Descripción |
|-----------|-----------------|-------------|
| Gain      | 0-100           | Controla la cantidad de distorsión/fuzz |
| Tone      | 0-100           | Ajusta el balance de frecuencias, de oscuro a brillante |
| Output    | 0-100           | Controla el nivel de salida del efecto |
