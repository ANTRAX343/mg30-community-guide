# Octave-Shift (WAH)

## Descripción
Efecto de wah combinado con octavador. Permite seleccionar si la octava es hacia arriba o hacia abajo.

## Parámetros

| Parámetro         | Valores posibles | Descripción                                 |
|-------------------|-----------------|---------------------------------------------|
| Switch            | Up / Down       | Selecciona octava arriba o abajo            |
| Pedal de expresión| 0–100           | Controla la apertura del filtro wah          |
