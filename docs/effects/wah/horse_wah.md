# Horse Wah (WAH)

## Descripción
Efecto wah con controles adicionales para esculpir el tono.

## Parámetros

| Parámetro         | Valores posibles | Descripción                                 |
|-------------------|-----------------|---------------------------------------------|
| Contour           | 0–100           | Ajusta la forma de la curva del wah         |
| Range             | 0–5             | Ajusta el rango de frecuencia del efecto    |
| Pedal de expresión| 0–100           | Controla la apertura del filtro wah         |
