# MOD: S.C.F.

## Emulación

- Modelo real: Stone Chorus Flanger
- Tipo: Chorus/Flanger híbrido
- Carácter: modulación versátil y rica

## Descripción

Efecto híbrido que combina características de chorus y flanger en un solo algoritmo. Ofrece modulaciones ricas y complejas que van desde chorus suaves hasta flanging dramático.

## Parámetros

### Speed
- Función: Velocidad de modulación
- Rango: 0 a 100
- Efecto: Velocidad del LFO modulador

### Width
- Función: Amplitud del efecto
- Rango: 0 a 100
- Efecto: Separación estéreo y amplitud

### Mode
- Función: Modo de operación
- Rango: Chorus/Flanger
- Efecto: Selector entre chorus y flanger

### Intensity
- Función: Intensidad del efecto
- Rango: 0 a 100
- Efecto: Profundidad de modulación
