# Rotary (MOD)

## Descripción
Falta complementar.

## Parámetros

### Balance
- Función: Balance entre woofer y tweeter
- Rango: 0 a 100
- Efecto: Mezcla entre altavoces graves y agudos

### Speed
- Función: Velocidad de rotación
- Rango: 0 a 100
- Efecto: Velocidad del efecto de altavoz giratorio
