# Flanger (MOD)

## Descripción
Falta complementar.

## Parámetros

### Depth
- Función: Profundidad del flanger
- Rango: Min a Max
- Efecto: Intensidad del efecto de barrido

### Rate
- Función: Velocidad de modulación
- Rango: Min a Max
- Efecto: Velocidad del LFO del flanger

### Width
- Función: Amplitud estéreo
- Rango: Min a Max
- Efecto: Separación estéreo del efecto

### Feedback
- Función: Realimentación
- Rango: Min a Max
- Efecto: Intensifica el efecto y añade resonancia
