# MOD: Phase 100

## Emulación

- Modelo real: MXR Phase 100
- Tipo: Phaser avanzado
- Carácter: phasing versátil y controlable

## Descripción

Emulación del MXR Phase 100, hermano mayor del Phase 90 con más controles y versatilidad. Ofrece mayor rango de sonidos y control sobre el efecto de phasing.

## Parámetros

### Intensity
- Función: Intensidad del efecto
- Rango: 0 a 100
- Efecto: Profundidad del phasing

### Speed
- Función: Velocidad de barrido del phaser
- Rango: 0 a 100
- Efecto: Velocidad de modulación del filtro
