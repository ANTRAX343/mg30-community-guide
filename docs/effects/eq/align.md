# EQ: Align

## Emulación

- Modelo real: EQ correctivo/alineación tonal
- Tipo: EQ correctivo
- Carácter: ajuste sutil para alineación tonal

## Descripción

Ecualizador diseñado para corregir y alinear el tono, ideal para ajustes sutiles que mejoren la respuesta general del instrumento.

## Parámetros

- HPF (High Pass Filter): 50 Hz, 80 Hz, 160 Hz (frecuencias de corte)
- VOLUME: -15 dB a +15 dB (nivel general de salida)
- 110 Hz: -15 dB a +15 dB (graves)
- 340 Hz: -15 dB a +15 dB (medios bajos)
- 660 Hz: -15 dB a +15 dB (medios)
- 1300 Hz: -15 dB a +15 dB (medios)
- 2600 Hz: -15 dB a +15 dB (medios altos)
- 5000 Hz: -15 dB a +15 dB (agudos/presencia)
