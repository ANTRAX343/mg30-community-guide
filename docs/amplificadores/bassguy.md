# BassGuy (AMP)

## Descripción

El modelo "BassGuy" está inspirado en el Ampeg SVT, un clásico de amplificación para bajo conocido por su potencia, graves profundos y presencia en bandas en vivo. Proporciona un low-end sólido y contundente.

En la MG30, "BassGuy" busca ofrecer ese carácter de bajo clásico, con controles para moldear graves y presencia.

## Parámetros

| Parámetro | Valores posibles | Descripción                                                                |
| --------- | ---------------- | -------------------------------------------------------------------------- |
| gain      | 0-100            | Controla la ganancia de entrada; desde limpio a saturación leve para bajo. |
| master    | 0-100            | Volumen general del amplificador/modelado.                                 |
| bass      | 0-100            | Control de frecuencias graves; añade cuerpo y profundidad.                 |
| middle    | 0-100            | Control de frecuencias medias; aporta definición y corte.                  |
| treb      | 0-100            | Control de frecuencias agudas; ajusta claridad y presencia.                |
| pres      | 0-100            | Presencia: ajuste de proyección en agudos.                                 |
| level     | 0-100            | Nivel de salida final del amplificador/emulación.                          |
