# DGLASS (AMP)

## Descripción

"DGLASS" es un modelo con enfoque en control de bandas medias y presencia; incorpora selecciones de frecuencia para moldear el carácter.

En la MG30, "DGLASS" permite seleccionar bandas medias y ajustar lo/hi mids para esculpir el tono en mezcla.

## Parámetros

| Parámetro  | Valores posibles | Descripción                                                  |
| ---------- | ---------------- | ------------------------------------------------------------ |
| master     | 0-100            | Volumen general del amplificador/modelado.                   |
| bass       | 0-100            | Control de frecuencias graves.                               |
| lo-mid-sel | 3 posiciones     | Selector para la banda baja-media: 250 Hz / 500 Hz / 1 kHz.  |
| lo_mids    | 0-100            | Ajuste de la banda baja-media.                               |
| hi_mids    | 0-100            | Ajuste de la banda alta-media.                               |
| hi-mid-sel | 3 posiciones     | Selector para la banda alta-media: 750 Hz / 1.5 kHz / 3 kHz. |
| treb       | 0-100            | Control de frecuencias agudas.                               |
| level      | 0-100            | Nivel de salida final del amplificador/emulación.            |
