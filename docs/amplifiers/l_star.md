# L Star (AMP)

## Descripción

"L Star" es un modelo orientado a respuesta equilibrada con controles completos de EQ y opciones de presencia/bias para moldear carácter.

En la MG30, "L Star" ofrece versatilidad para clean y overdrive, con control de presencia y bias para ajustar compresión y brillo.

## Parámetros

| Parámetro | Valores posibles | Descripción                                            |
| --------- | ---------------- | ------------------------------------------------------ |
| gain      | 0-100            | Controla la ganancia de entrada.                       |
| master    | 0-100            | Volumen general del amplificador/modelado.             |
| bass      | 0-100            | Control de frecuencias graves.                         |
| middle    | 0-100            | Control de frecuencias medias.                         |
| treb      | 0-100            | Control de frecuencias agudas.                         |
| pres      | 0-100            | Presencia: ajuste de proyección en agudos.             |
| bias      | 0-100            | Polarización virtual que afecta compresión y carácter. |
| level     | 0-100            | Nivel de salida final del amplificador/emulación.      |
