# MLD (AMP)

## Descripción

El modelo "MLD" emula un amplificador orientado a medios definidos; ofrece desde tonos limpios a saturaciones moderadas con énfasis en la banda media.

En la MG30, "MLD" es útil para recortar mezcla y obtener presencia en solistas y ritmos con cuerpo.

## Parámetros

| Parámetro | Valores posibles | Descripción                                              |
| --------- | ---------------- | -------------------------------------------------------- |
| gain      | 0-100            | Controla la ganancia de entrada; de limpio a saturación. |
| master    | 0-100            | Volumen general del amplificador/modelado.               |
| bass      | 0-100            | Control de frecuencias graves.                           |
| middle    | 0-100            | Control de frecuencias medias; define presencia.         |
| m freq    | 50-5000 Hz       | Frecuencia central del control de medios.                |
| treb      | 0-100            | Control de frecuencias agudas.                           |
| level     | 0-100            | Nivel de salida final del amplificador/emulación.        |
