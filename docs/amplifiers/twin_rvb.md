# Twin RVB (AMP)

## Descripción

El modelo "Twin RVB" está inspirado en el Fender Twin Reverb, un amplificador de válvulas legendario conocido por su limpieza cristalina, gran headroom y un reverb de muelles muy presente. Es ideal para estilos que requieren un limpio amplio y definido; al empujar sus etapas puede mantener claridad sin perder definición.

## Parámetros

| Parámetro | Valores posibles | Descripción |
|-----------|-----------------|-------------|
| gain      | 0-100           | Controla la ganancia de entrada; en ajustes bajos mantiene el limpio característico y en ajustes altos introduce más cuerpo y ligera compresión. |
| master    | 0-100           | Volumen general del amplificador/modelado. |
| bass      | 0-100           | Controla las frecuencias graves. |
| middle    | 0-100           | Controla las frecuencias medias. |
| treb      | 0-100           | Controla las frecuencias agudas. |
| Bright    | On/Off (switch) | Activa un filtro o énfasis en agudos para mayor claridad/presencia (switch). |
| bias      | 0-100           | Ajusta la polarización virtual para cambiar el carácter tonal y la respuesta dinámica. |
| level     | 0-100           | Nivel de salida final del amplificador/emulación. |
