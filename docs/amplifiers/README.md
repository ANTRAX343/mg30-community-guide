## Documentation Format

Each amplifier includes:
- **Description** - Tone character and real-world inspiration
- **Parameters** - All available controls (gain, master, EQ, etc.)
- **Sweet Spots** - Recommended settings for different styles
- **Compatible Effects** - Which effects work best with each amp
- **Famous Users** - Artists known for using the real amplifier
