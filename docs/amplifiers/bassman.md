
# Bassmate (AMP)

## Descripción

El modelo Bassmate está basado en el amplificador Bassmate, un combo de válvulas vintage conocido por su sonido cálido, respuesta dinámica y carácter clásico. Aunque menos famoso que otros modelos, el Bassmate es apreciado por su tono suave y musical, ideal para blues, rock y estilos vintage.

## Parámetros

| Parámetro | Valores posibles | Descripción |
|-----------|-----------------|-------------|
| Gain      | 0-100           | Controla la ganancia de entrada del preamplificador |
| Master    | 0-100           | Ajusta el volumen general del amplificador |
| Bass      | 0-100           | Controla la cantidad de frecuencias graves |
| Middle    | 0-100           | Controla la cantidad de frecuencias medias |
| Treble    | 0-100           | Controla la cantidad de frecuencias agudas |
| Presence  | 0-100           | Ajusta la presencia/agudos para mayor definición |
| Bias      | 0-100           | Ajusta la polarización de las válvulas para modificar el carácter tonal |
| Level     | 0-100           | Controla el nivel de salida final del amplificador |
