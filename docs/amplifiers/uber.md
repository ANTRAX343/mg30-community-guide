# UBER (AMP)

## Descripción

El modelo "UBER" está inspirado en el Bogner Uberschall, un amplificador alemán de alta ganancia diseñado para producir tonos pesados y densos con gran sustain; es un referente para sonidos modernos y metal por su cuerpo y enfoque en medios/bajos.

En la MG30, "UBER" reproduce ese high-gain contundente, permitiendo ajustes finos mediante `pres` y `bias` para controlar proyección y compresión.

## Parámetros

| Parámetro | Valores posibles | Descripción                                                     |
| --------- | ---------------- | --------------------------------------------------------------- |
| gain      | 0-100            | Controla la ganancia de entrada; de crunch a high-gain extremo. |
| master    | 0-100            | Volumen general del amplificador/modelado.                      |
| bass      | 0-100            | Control de frecuencias graves; aporta peso.                     |
| middle    | 0-100            | Control de frecuencias medias; define presencia y corte.        |
| treb      | 0-100            | Control de frecuencias agudas; añade definición.                |
| pres      | 0-100            | Presencia: ajuste de proyección en agudos.                      |
| bias      | 0-100            | Polarización virtual que afecta compresión y carácter.          |
| level     | 0-100            | Nivel de salida final del amplificador/emulación.               |
