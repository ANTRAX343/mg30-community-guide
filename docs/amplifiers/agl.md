# AGL (AMP)

## Descripción

El modelo "AGL" emula un cabezal de bajo Aguilar inspirado en el Tone Hammer 700 (Tone Hammer 700 Bass Amp) y en la MG30 representa una referencia para tonos de bajo con cuerpo y claridad. Está orientado a preservar el ataque y la definición en las frecuencias graves.

En la MG30, "AGL" ofrece control preciso de EQ y rango medio mediante la sección `m freq` para esculpir el tono en mezcla.

## Parámetros

| Parámetro | Valores posibles | Descripción                                                                    |
| --------- | ---------------- | ------------------------------------------------------------------------------ |
| gain      | 0-100            | Controla la ganancia de entrada; define desde limpio a saturación leve.        |
| master    | 0-100            | Volumen general del amplificador/modelado.                                     |
| bass      | 0-100            | Control de frecuencias graves; añade cuerpo y pegada.                          |
| middle    | 0-100            | Control de frecuencias medias; importante para el cuerpo.                      |
| m freq    | 50-5000 Hz       | Frecuencia central del control de medios (m freq) para ajustar la banda media. |
| treb      | 0-100            | Control de frecuencias agudas; aporta claridad y definición.                   |
| level     | 0-100            | Nivel de salida final del amplificador/emulación.                              |
