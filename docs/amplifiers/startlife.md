# Startlife (AMP)

## Descripción

"Startlife" es un modelo con opciones de contorno tonal (vintage/modern) pensado para variar carácter entre cálido y brillante.

En la MG30, "Startlife" ofrece un control de medios preciso y un switch de contorno para elegir el carácter general.

## Parámetros

| Parámetro | Valores posibles | Descripción                                       |
| --------- | ---------------- | ------------------------------------------------- |
| bass      | 0-100            | Control de frecuencias graves.                    |
| midd      | 0-100            | Control de frecuencias medias.                    |
| mid freq  | 50-5000 Hz       | Frecuencia central del control de medios.         |
| treb      | 0-100            | Control de frecuencias agudas.                    |
| contour   | 3 posiciones     | Switch de contorno: Vintage / Off / Modern.       |
| volume    | 0-100            | Volumen/preamp del canal.                         |
| level     | 0-100            | Nivel de salida final del amplificador/emulación. |
