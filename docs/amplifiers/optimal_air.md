# Optimal Air (AMP)

## Descripción

"Optimal Air" es un modelo orientado a claridad y dinámica, pensado para clean y overdrive transparente con respuesta abierta en agudos.

En la MG30, "Optimal Air" aporta detalle y separación entre notas, útil en ritmos arpegiados y acordes definidos.

## Parámetros

| Parámetro | Valores posibles | Descripción                                       |
| --------- | ---------------- | ------------------------------------------------- |
| gain      | 0-100            | Control de la ganancia de entrada.                |
| master    | 0-100            | Volumen general del amplificador/modelado.        |
| bass      | 0-100            | Control de frecuencias graves.                    |
| middle    | 0-100            | Control de frecuencias medias.                    |
| treb      | 0-100            | Control de frecuencias agudas.                    |
| level     | 0-100            | Nivel de salida final del amplificador/emulación. |
