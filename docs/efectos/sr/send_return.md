# S/R: Send & Return

## Descripción

Loop de efectos externos que permite insertar pedales o procesadores adicionales en la cadena de señal de la MG30. Ubicado entre el bloque EQ y el bloque RVB.

## Parámetros

### Send

- Rango: -15 dB a +15 dB
- Función: Controla el nivel de señal enviada a los efectos externos
- Uso: Ajustar según la sensibilidad del efecto externo

### Return

- Rango: -15 dB a +15 dB
- Función: Controla el nivel de señal que retorna de los efectos externos
- Uso: Balancear el nivel del efecto procesado con la señal principal

### Routing

- **Branch (Paralelo)**: La señal se divide, una parte va al loop externo y otra continúa directo
- **Serial (Serie)**: Toda la señal pasa por el loop externo antes de continuar

## Conectividad

### Conexiones Físicas

- **Send Output**: Conector de salida hacia el efecto externo
- **Return Input**: Conector de entrada desde el efecto externo
- **Cables**: Usar cables de guitarra estándar TS 1/4"

## Casos de Uso

### Efectos Recomendados para el Loop

- Delays analógicos vintage
- Reverbs de alta calidad
- Efectos de modulación especiales
- Procesadores de rack
- Pedales boutique específicos

### Configuraciones Típicas
- **Branch**: Para efectos de tiempo (delay/reverb) en paralelo
- **Serial**: Para efectos de ganancia o modulación que deben procesar toda la señal

## Tips de Uso

1. **Niveles**: Comenzar con ambos controles en 0 dB y ajustar según necesidad
2. **Impedancia**: Verificar compatibilidad con efectos externos
3. **Ruido**: Usar cables de calidad para minimizar ruido en el loop
4. **Bypass**: El loop se puede bypass completamente si no hay nada conectado
