
# ST Singer (EFX)

## Descripción
El pedal Steel Singer Drive de la serie Reissue de NUX es un overdrive analógico con el carácter tonal del famoso amplificador boutique de California (Dumble Steel String Singer). Presenta un high end muy suave y un rango medio dulce. Puede usarse como overdrive principal o para empujar un amplificador, logrando que la guitarra "cante" y manteniendo claridad y musicalidad incluso con ganancia elevada.

## Parámetros

| Parámetro | Valores posibles | Descripción |
|-----------|-----------------|-------------|
| Volume    | 0-100           | Controla el nivel de salida del efecto |
| Gain      | 0-100           | Ajusta la cantidad de saturación/overdrive |
| Filter    | 0-100           | Controla el recorte de frecuencias altas para moldear el tono |
