# IR: USER COLLECTION

## Emulación

- Modelo real: N/A (colección del usuario)
- Tipo: ranuras para IR personalizados

## Descripción

Permite cargar IRs de terceros o capturas propias. Útil para adaptar el sonido a salas específicas, equipos particulares o preferencias personales.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
