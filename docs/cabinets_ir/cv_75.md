# IR: CV-75

## Emulación

- Modelo real: Celestion CV-75 1x12"
- Tipo de gabinete: 1x12"
- Carácter: moderno, potente, agudos controlados

## Descripción

Altavoz moderno con buena tolerancia al volumen y agudos suaves, ideal para high-gain sin excesivo filo.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
