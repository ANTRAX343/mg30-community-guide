# IR: AGL DB810

## Emulación

- Modelo real: Aguilar DB810 8x10" (bajo)
- Tipo de gabinete: 8x10"
- Carácter: graves masivos, presencia definida

## Descripción

Gabinete masivo para bajo con gran capacidad de graves y presencia clara en medios, ideal para estilos que requieren potencia y definición.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
