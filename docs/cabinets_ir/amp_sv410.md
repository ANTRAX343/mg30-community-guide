# IR: AMP SV410

## Emulación

- Modelo real: Ampeg SVT 4x10" (bajo)
- Tipo de gabinete: 4x10"
- Carácter: ataque rápido, medios marcados

## Descripción

Versión más compacta del SVT con respuesta rápida y medios definidos. Excelente equilibrio entre definición y cuerpo para bajo.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
