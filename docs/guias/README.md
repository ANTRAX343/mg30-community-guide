# Guías Prácticas — NUX MG-30 v5.0.2

> **Firmware objetivo:** NUX MG-30 **v5.0.2**

Guías orientadas a escenarios reales de uso de la NUX MG-30. Cada guía cubre configuración, ajustes recomendados y consejos prácticos.

## Guías disponibles

### [🔊 Directo a PA](./directo_a_pa.md)

Cómo conectar la MG-30 directamente a una mesa de mezclas o sistema de PA para uso en vivo sin amplificador. Configuración de salida XLR, ajuste de niveles y gabinetes recomendados.

### [🔁 Conexión 4 Cable Method (4CM)](./4cm.md)

Cómo usar la MG-30 con un amplificador real mediante el método de 4 cables, aprovechando el loop de efectos del amplificador para separar efectos pre/post-amplificador.

### [💻 Grabación por USB](./grabacion_usb.md)

Cómo usar la MG-30 como interfaz de audio USB para grabar directamente en un DAW (Reaper, GarageBand, Ableton, etc.). Configuración del driver, latencia y ajustes de nivel.

### [📊 Niveles y Clipping](./niveles_y_clipping.md)

Cómo gestionar correctamente los niveles de señal en la cadena para evitar distorsión no deseada (clipping) y mantener un nivel de salida consistente entre presets.

### [🔇 Ruido y Noise Gate](./ruido_gate.md)

Cómo identificar y reducir el ruido de fondo, ajuste del Noise Gate (GATE), técnicas para configuraciones de alta ganancia y solución de problemas comunes de ruido.
