# IR: AMP SV810

## Emulación

- Modelo real: Ampeg SVT 8x10" (bajo)
- Tipo de gabinete: 8x10"
- Carácter: clásico SVT, medios presentes, punch

## Descripción

El clásico 8x10" de Ampeg, estándar en rock y metal para bajo. Medios presentes con gran pegada y carácter SVT inconfundible.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
