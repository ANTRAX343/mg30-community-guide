# IR: JZ120

## Emulación

- Modelo real: Roland JC-120 Jazz Chorus
- Tipo de gabinete: 2x12"
- Carácter: limpio, cristalino, estéreo, agudos brillantes

## Descripción

IR basado en el clásico JC-120. Ideal para cleans, chorus y sonidos comprimidos con gran definición en agudos.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz

## Micrófonos disponibles

1. DYN421 (dinámico)
2. S57 (Shure SM57, dinámico)
3. U87 (condensador)
4. R122 (cinta)
5. R121 (cinta)
6. C414 (condensador)
7. C3000 (condensador)
8. B52 (dinámico/grave)
