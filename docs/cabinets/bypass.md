# IR: BYPASS

## Emulación

- Modelo real: N/A (sin simulación de gabinete)
- Tipo de gabinete: N/A
- Carácter: señal directa sin coloración

## Descripción

Desactiva la simulación de gabinete. La señal pasa sin procesamiento de IR, útil cuando se usa gabinete real o IR externo.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
