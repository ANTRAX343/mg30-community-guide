# IR: BASSGUY 410

## Emulación

- Modelo real: Ampeg SVT 410 (bajo)
- Tipo de gabinete: 4x10"
- Carácter: clásico, balanceado

## Descripción

Configuración clásica Ampeg 4x10" con balance tradicional entre graves y medios, opción versátil para múltiples estilos.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
