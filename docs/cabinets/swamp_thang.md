# IR: SWAMP THANG

## Emulación

- Modelo real: Eminence Swamp Thang 1x12"
- Tipo de gabinete: 1x12"
- Carácter: alto rendimiento, graves fuertes, limpio contundente

## Descripción

Altavoz de alta potencia con graves sólidos y respuesta limpia, ideal para tonos que requieren claridad y pegada.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
