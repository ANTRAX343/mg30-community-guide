# IR: AMP SV212

## Emulación

- Modelo real: Ampeg SVT 2x12" (bajo)
- Tipo de gabinete: 2x12"
- Carácter: más redondo que 4x10", mantiene definición

## Descripción

Configuración 2x12" con carácter más suave que los 4x10", ideal para estilos que requieren menos ataque y más cuerpo.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
