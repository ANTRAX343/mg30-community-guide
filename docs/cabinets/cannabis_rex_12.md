# IR: CANNABIS REX 12

## Emulación

- Modelo real: Eminence Cannabis Rex 1x12"
- Tipo de gabinete: 1x12"
- Carácter: medios cálidos, agudos suaves

## Descripción

Altavoz con respuesta suave en agudos y medios cálidos, ideal para domar rigs brillantes y aportar calidez.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
