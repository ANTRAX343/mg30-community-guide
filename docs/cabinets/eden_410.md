# IR: EDEN 410

## Emulación

- Modelo real: Eden D410XLT 4x10" (bajo)
- Tipo de gabinete: 4x10"
- Carácter: medios altos con presencia, graves apretados

## Descripción

Gabinete Eden con medios altos prominentes y graves controlados, ideal para cortar en la mezcla sin perder cuerpo.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
