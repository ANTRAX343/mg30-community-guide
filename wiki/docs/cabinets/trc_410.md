# IR: TRC 410

## Emulación

- Modelo real: Trace Elliot 1048H 4x10" (bajo)
- Tipo de gabinete: 4x10"
- Carácter: brillantez controlada, pegada

## Descripción

Gabinete británico para bajo con carácter brillante pero controlado, excelente pegada para estilos modernos.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
