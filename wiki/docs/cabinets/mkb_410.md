# IR: MKB 410

## Emulación

- Modelo real: Markbass Standard 104HR 4x10" (bajo)
- Tipo de gabinete: 4x10"
- Carácter: moderno, eficiente, centrado en medios

## Descripción

Gabinete moderno para bajo con eficiencia alta y enfoque en medios, ideal para estilos contemporáneos como slap y fingerstyle.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
