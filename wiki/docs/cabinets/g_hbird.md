# IR: G-HBIRD

## Emulación

- Modelo real: Gibson Hummingbird (IR acústica)
- Tipo: IR de guitarra acústica
- Carácter: cálido, con cuerpo

## Descripción

IR del Gibson Hummingbird con carácter cálido y cuerpo completo, excelente para strumming y acompañamiento acústico.

## Parámetros

- level: -12 dB a +12 dB (por defecto 0 dB)
- low_cut: 20 Hz a 1 kHz
- high_cut: 5 kHz a 20 kHz
