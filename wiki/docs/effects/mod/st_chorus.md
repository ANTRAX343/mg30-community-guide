# MOD: ST.Chorus

## Emulación

- Modelo real: Stereo Chorus
- Tipo: Chorus estéreo
- Carácter: chorus estéreo espacial

## Descripción

Efecto de chorus estéreo que crea una modulación rica y espacial. Diseñado específicamente para aprovechar el campo estéreo, generando un efecto envolvente y dimensional.

## Parámetros

### Intensity
- Función: Intensidad del efecto chorus
- Rango: 0 a 100
- Efecto: Controla la profundidad del chorus

### Width
- Función: Amplitud estéreo
- Rango: 0 a 100
- Efecto: Controla la separación estéreo del efecto

### Rate
- Función: Velocidad de modulación
- Rango: 0 a 100
- Efecto: Velocidad de oscilación del chorus
