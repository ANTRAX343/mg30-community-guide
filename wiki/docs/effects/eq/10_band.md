# EQ: 10-Band

## Emulación

- Modelo real: Ecualizador gráfico de 10 bandas
- Tipo: EQ gráfico
- Carácter: control visual por bandas de frecuencia

## Descripción

Ecualizador gráfico estilo clásico con 10 bandas fijas, ideal para ajustes rápidos y visuales del espectro de frecuencias.

## Parámetros

- VOL: -12 dB a +12 dB (volumen general de salida)
- 31.25 Hz: -12 dB a +12 dB (graves muy profundos)
- 62.5 Hz: -12 dB a +12 dB (graves profundos)
- 125 Hz: -12 dB a +12 dB (graves)
- 250 Hz: -12 dB a +12 dB (graves/medios bajos)
- 500 Hz: -12 dB a +12 dB (medios bajos)
- 1K Hz: -12 dB a +12 dB (medios)
- 2K Hz: -12 dB a +12 dB (medios)
- 4K Hz: -12 dB a +12 dB (medios altos)
- 8K Hz: -12 dB a +12 dB (agudos)
- 16K Hz: -12 dB a +12 dB (agudos muy altos)
- GAIN: -12 dB a +12 dB (ganancia general)
