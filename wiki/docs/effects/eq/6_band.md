# EQ: 6-Band

## Emulación

- Modelo real: Ecualizador paramétrico de 6 bandas
- Tipo: EQ paramétrico
- Carácter: control preciso de frecuencias clave

## Descripción

Ecualizador de 6 bandas con frecuencias fijas optimizadas para guitarra, permite ajuste fino de las frecuencias más importantes del espectro tonal.

## Parámetros

- level: -15 dB a +15 dB (nivel general de salida)
- 100 Hz: -15 dB a +15 dB (graves profundos)
- 220 Hz: -15 dB a +15 dB (graves/medios bajos)
- 500 Hz: -15 dB a +15 dB (medios bajos)
- 1.2 kHz: -15 dB a +15 dB (medios)
- 2.6 kHz: -15 dB a +15 dB (medios altos)
- 6.4 kHz: -15 dB a +15 dB (agudos/presencia)
