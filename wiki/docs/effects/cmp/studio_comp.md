# Studio Comp (CMP)

## Descripción
Compresor de estudio profesional, diseñado para ofrecer control detallado sobre la dinámica de la señal. Permite ajustar el umbral (threshold), la relación de compresión (ratio), la ganancia de salida (gain) y el tiempo de liberación (release), logrando una compresión precisa y transparente ideal para aplicaciones musicales exigentes.

## Parámetros

| Parámetro | Valores posibles | Descripción |
|-----------|-----------------|-------------|
| Thr (Threshold) | 0–100 | Define el nivel a partir del cual se activa la compresión. |
| Ratio | 0–100 | Relación de compresión aplicada a la señal que supera el umbral. |
| Gain | 0–100 | Ajusta la ganancia de salida tras la compresión. |
| Release | 0–100 | Controla el tiempo que tarda la compresión en dejar de actuar tras bajar del umbral. |
