# Stageman (AMP)

## Descripción

"Stageman" es un modelo orientado a uso en directo: ofrece headroom y una ecualización simple y efectiva para cortar en mezcla.

En la MG30, "Stageman" sirve como un amplificador versátil para escenarios pequeños y medianos.

## Parámetros

| Parámetro | Valores posibles | Descripción                                       |
| --------- | ---------------- | ------------------------------------------------- |
| gain      | 0-100            | Controla la ganancia de entrada.                  |
| master    | 0-100            | Volumen general del amplificador/modelado.        |
| bass      | 0-100            | Control de frecuencias graves.                    |
| middle    | 0-100            | Control de frecuencias medias.                    |
| treb      | 0-100            | Control de frecuencias agudas.                    |
| level     | 0-100            | Nivel de salida final del amplificador/emulación. |
