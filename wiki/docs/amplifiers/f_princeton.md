# F Princeton (AMP)

## Descripción

"F Princeton" emula la sensación y respuesta de un Princeton clásico: dinámica, sensibilidad al ataque y tono cristalino en limpio.

En la MG30, "F Princeton" es ideal para tonos limpios y overdrive suave.

## Parámetros

| Parámetro | Valores posibles | Descripción                                            |
| --------- | ---------------- | ------------------------------------------------------ |
| volume    | 0-100            | Volumen del canal.                                     |
| treb      | 0-100            | Control de frecuencias agudas.                         |
| bass      | 0-100            | Control de frecuencias graves.                         |
| bias      | 0-100            | Polarización virtual que afecta carácter y compresión. |
| level     | 0-100            | Nivel de salida final del amplificador/emulación.      |
